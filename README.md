```bash

cd create-a-chatgpt-style-app-shell-left-ch
npm install
cp .env.example .env.local
# set DATABASE_URL (and OpenAI keys if using the agent route)
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
